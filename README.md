# Personal Portfolio

## Project Overview  
This project is a modern, responsive personal portfolio website developed using HTML and CSS. It is designed to present my academic background, technical skills, projects, and certifications in a structured and visually appealing manner.

## Repository Link

## About Me  
I am Sneha Jayamohanan, currently pursuing a B.Tech in Artificial Intelligence and Data Science (Semester 4) at Amrita Vishwa Vidyapeetham, Amritapuri. My interests include machine learning, computer vision, and building data-driven intelligent systems.

---

## Technologies Used  
- HTML5 – for semantic structuring of content  
- CSS3 – for styling and layout, including:  
  - Flexbox and Grid  
  - Responsive design using media queries  
  - Animations and transitions  

---

## Features  
- Clean and modern user interface with a dark theme  
- Fully responsive design compatible with different screen sizes  
- Smooth transitions and hover effects  
- Organized sections for projects, skills, and certifications  
- Dedicated project descriptions with relevant details
