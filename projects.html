Dementia Detection via Eye Tracking

PythonPyGameComputer Vision

This project aims to analyze image data, specifically focusing on eye-related features for potential use in dementia detection. The algorithm detects the pupil and iris through image processing, calculating specific features like dilation even in challenging conditions like glare or partial occlusion. The application includes user-friendly instructions for beginners and a clear interface to view the calculated results via a stimulus engine built with PyGame.
